MDWeather
========
一个基于material design的天气app<br/>
网络请求采用retrofit 2.0 + rxjava + eventbus，自定义了多种view用于展示七天天气预报，空气质量及日出日落。
### 数据来源
[天气查询](http://apistore.baidu.com/apiworks/servicedetail/112.html "百度的api store")  百度的api store中的一个关于天气的api
